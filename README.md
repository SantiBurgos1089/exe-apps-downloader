# exe-apps-downloader

Download Windows apps (exe, msi, others) from a site or URL given

## Requirements

### Frontend

* GTK4
* libAdapta or libAdwaita
* Python >= 3.10 (using Debian 13 as a base)

### Backend

* requests
* GObject
* GLib
* libnotify
* Python >= 3.10 (using Debian 13 as a base)
* [xapp-symbolic-icons](https://github.com/xapp-project/xapp-symbolic-icons)
